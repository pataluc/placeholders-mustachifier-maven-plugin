placeholders-mustachifier:mustache-tovalue

Full name:

com.xebialabs.utils:placeholders-mustachifier-maven-plugin:0.5:mustache-tovalue

Description:

From {{key:value}} to value

Attributes:

  • Requires a Maven project to be executed.
  • The goal is thread-safe and supports parallel builds.
  • Binds by default to the lifecycle phase: compile.

Optional Parameters

Name Type Since Description
delimiters String - The delimiter format
Default value is: {{ }}.
files List - List of SingleFile to be scanned. Must be text file, archive not supported.
filesets List - FileSet including special entries
valueProvider ValueProviderConfiguration - The separator format from the placeholder name and the default value
valueSeparator String - The separator format from the placeholder name and the default value
Default value is: :.

Parameter Details

delimiters:

The delimiter format
  • Type: java.lang.String
  • Required: No
  • Default: {{ }}

files:

List of SingleFile to be scanned. Must be text file, archive not supported.
  • Type: java.util.List
  • Required: No

filesets:

FileSet including special entries
  • Type: java.util.List
  • Required: No

valueProvider:

The separator format from the placeholder name and the default value
  • Type: com.xebialabs.maven.mustache.ValueProviderConfiguration
  • Required: No

valueSeparator:

The separator format from the placeholder name and the default value
  • Type: java.lang.String
  • Required: No
  • Default: :
